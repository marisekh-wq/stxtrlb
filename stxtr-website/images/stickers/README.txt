DROP YOUR STICKER IMAGES HERE
==============================
Accepted formats: PNG (recommended, transparent bg), JPG, WEBP, SVG
Recommended size: 200x200 px, square

Name your files to match the sticker IDs in index.html:

CUTE:       cat.png  bunny.png  bear.png  unicorn2.png  frog.png  fox2.png  bee2.png  duck.png
AESTHETIC:  moon2.png  star2.png  sparkle.png  orb2.png  diamond.png  rainbow2.png  magic.png  planet2.png
NATURE:     blossom.png  hibiscus2.png  sunflr.png  strawb.png  waterml.png  leaf2.png  wave2.png  mushroom2.png
WORDS:      word stickers use coloured text — no image needed, but you can add one too.

Then in index.html, find the STICKERS array and set img: to the file path.
Example:
  { id:'cat', e:'🐱', n:'Kitty', cat:'cute', img: 'images/stickers/cat.png' },
