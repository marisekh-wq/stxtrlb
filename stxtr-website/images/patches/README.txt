DROP YOUR PATCH IMAGES HERE
============================
Accepted formats: PNG (recommended), JPG, WEBP, SVG
Recommended size: 200x200 px, square, transparent background (PNG)

Name your files to match the patch IDs in index.html:

  cherry.png       star.png        butterfly.png   rainbow.png
  heart.png        hibiscus.png    mushroom.png    moon.png
  sun.png          unicorn.png     leaf.png        crystal.png
  bow.png          strawberry.png  wave.png        orb.png
  fox.png          bee.png         sunflower.png   music.png
  flamingo.png     watermelon.png  glowstar.png    planet.png

Then in index.html, find the PATCHES array and set img: to the file path.
Example:
  { id:'cherry', e:'🌸', n:'Cherry Blossom', img: 'images/patches/cherry.png' },
